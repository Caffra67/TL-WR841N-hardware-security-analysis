# TL-WR841N Hardware Security Analysis

Hardware security analysis of a TP-Link TL-WR841N router, focused on identifying physical and firmware-level attack surfaces in embedded Linux devices.

This repository documents a hands-on hardware security study of the TP-Link TL-WR841N router, with emphasis on understanding embedded Linux attack surfaces and generalizable methodologies.
This project is inspired by structured hardware hacking training from TCM Security. No proprietary course materials are included.

## Device under test (DUT)

- TL-WR841N

## Device

- Multimeter
- KeeYees SOP8/SOIC8 Test Clip + CH341A USB Programmer
- Comidox USB Logic Analyzer 24MHz 8CH
- IZOKEE CP2102 Module USB to TTL

## Tools

- PulseView
- Nmap
- flashrom

Device owned by the author. No client data involved.
