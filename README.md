# TL-WR841N Hardware Security Analysis

Learning-focused hardware security analysis of a TP-Link TL-WR841N router, based on structured training scenarios and public techniques.

This repository documents a hands-on hardware security study of the TP-Link TL-WR841N router, with emphasis on understanding embedded Linux attack surfaces and generalizable methodologies.
This project is inspired by structured hardware hacking training from TCM Security. No proprietary course materials are included.
